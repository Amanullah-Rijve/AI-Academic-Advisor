# 🎓 Smart Academic Advisor (DIU AI Competition)

AI-powered academic assistant for students.

## 🚀 Features
- AI-based advising chatbot
- Student-specific suggestions
- Course & CGPA guidance
- Modern UI (glassmorphism)

## 🛠 Tech Stack
- React.js
- Node.js (Express)
- FastAPI (Python)
- OpenAI / Grok API
- MySQL

## 🎯 Goal
Improve student academic decision-making using AI.
